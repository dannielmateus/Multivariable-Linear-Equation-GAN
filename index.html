Machine Learning to solve equations of multiple variables
By Daniel Mateus

This equation solver works with equations in the form

N = ax + by + cz + ...

Where x, y, z, ... are variables, and a, b, c, ... are constants.
It can take as many arguments as you want, and gives as a result
the possible solutions (there are infinite solutions for multivariable equations!)

    Generative Adversarial Network (GAN)

    This model starts by assigning the variables x, y, z, ... to random weights,
    and checking the loss L = N - (a*x + b*y + c*y + ...).
    Then it changes the weights (subtract or add some small random value)
    and see if the loss gets smaller. If it does, it continues, otherwise it
    ignores the change. It does that until the loss is smaller than the choosen maximum loss
